## 1. 方案定位与亮点（同 v1，简述）

与常见同题组的差异化核心：

1. 测点布置由**灵敏度分析驱动**，而非经验均布。
2. 失调诊断用 **"规则 HII 指数 + AI 分类器"双通道**，冗余容错。
3. 误差分析用 **Monte Carlo 传递（JCGM 101 标准）**，输出 HII 置信区间。
4. 支路流量测量做 **外夹超声 vs 插入电磁** 的严谨方案比选，并在现场设计"在线比对校核链"。

### 3.4 基于灵敏度分析的测点优化（亮点 1）

方法：

1. 用自写 Hardy-Cross 建立 4–6 支路数字孪生。
2. 对每条支路施加 ±20% 阻力扰动，记录每个候选测点的响应幅值 Δyⱼ / Δxᵢ → 灵敏度矩阵 S (n_meas × n_fault)。
3. 选择使 S 矩阵**列满秩** 且 **最小奇异值最大化** 的最小测点子集（贪心算法即可）。
4. 对选中的测点再做 1 处冗余（故障容错）。

结论预估：5 条支路场景下，**3 个支路流量 + 2 个支路压差** 即可唯一区分所有单点失调故障，其余支路只需温度采集即可（温度用于热量校核）。这就解释了"为什么不是每条支路都装电磁流量计"——**可观测性不需要那么多**。

这套方法论对答辩非常友好：对手若问"为什么测点这么布"，可以拿出灵敏度矩阵和 SVD 条件数曲线作答。

**最终决策**：

**答辩防御点**：若对方问"为何不全部用电磁"，核心回答是：
1. 题目约束明确"不允许大规模改造原有管网结构"—— 5 条支路全部开孔不满足。
2. HII ≥ 0.1 的判别阈值与 2% 流量误差不冲突，ROC 分析见误差分析章节。
3. 现场维护条件一般，一体式超声 10 年免维护，TCO 低于电磁。

## 5. 水力失调指标与诊断算法（v2 细化）

### 5.1 HII 综合水力失调指数（规则通道）

基础指标（全部归一化为无量纲）：

```
θ_Q,i = (Q_i − Q_i,设计) / Q_i,设计             # 支路流量偏差
θ_P   = std{ΔP_i} / mean{ΔP_i}                   # 支路压差分散度
θ_T,i = (ΔT_i − ΔT̄) / ΔT̄,  ΔT_i = T_sup,i − T_ret,i   # 温差一致性
```

综合指数：

```
HII = w_Q · max_i|θ_Q,i|  +  w_P · θ_P  +  w_T · max_i|θ_T,i|
```

权重建议 w_Q = 0.5, w_P = 0.3, w_T = 0.2（流量最直接，温差最慢最滞后）。

判定分级：

| HII 范围 | 状态 | 运行建议 |
|----------|------|----------|
| < 0.10 | 正常 | 继续监视 |
| 0.10 – 0.20 | 轻度失调 | 记录，下次巡检复查 |
| 0.20 – 0.35 | 中度失调 | 调节阀门/循环泵频率 |
| ≥ 0.35 | 重度失调 | 启动诊断 → AI 定位支路 |

### 5.2 AI 故障定位（增强通道）

**训练数据**（完全由数字孪生仿真产生）：

| 故障类型 | 样本量 | 扰动参数 |
|----------|--------|----------|
| 正常运行（多总流量工况） | 1500 | Q₀ 在 30–80 m³/h 均匀采样 |
| 单支路阻力+（堵塞） | 1000 | 阻力 ×1.2~5.0 随机支路 |
| 单支路阻力-（阀过开） | 1000 | 阻力 ×0.3~0.8 |
| 双支路同时失调 | 800 | 随机两条 |
| 循环泵频率偏移 | 400 | 频率 ±20% |
| 测量噪声工况 | 300 | 在正常上叠加 2% 噪声，用于鲁棒性 |

**特征向量（20 维）**：
```
x = [Q₀, {Q_i/Q₀}_{i=1..5}, ΔP, {ΔT_i}_{i=1..5}, {θ_i(阀位)}_{i=1..5}]
```

**模型**：
- Baseline：RandomForest（n_estimators=200, max_depth=12）
- 主力：XGBoost（max_depth=6, lr=0.1, n_estimators=300）
- 输出：故障类型分类（6 类）+ 失调支路编号（回归 → 1~5）

**评估**：80/20 split + 5-fold CV；指标 F1 macro、混淆矩阵、SHAP summary。
预期性能（基于文献类比）：F1 ≥ 0.95，远高于规则通道。

**关键防御点**：AI 训练数据是**仿真生成的**，教师可能问"仿真数据训练能否推广到实际"。回答：
1. 仿真参数已按题目给定工况覆盖，物理一致性由 Hardy-Cross 守恒律保证。
2. 我们同时保留 HII 规则通道，AI 失效时系统仍然可诊断，**不是单点失败的设计**。
3. 实际部署时可用现场几周正常数据做 **domain adaptation**（这是未来工作，不在作业范围）。

### 5.3 动态响应处理

- **EWMA 滑动滤波**：α=0.2，窗长 120 s。
- **工况归一化**：所有流量按 Q₀ 归一化（支路负荷变化导致的"伪失调"被消除）。
- **稳态判定**：Q₀ 的 5 min σ/μ < 3% 才判定进入稳态，否则挂起诊断。
- **温度反卷积**（加分项）：Pt100 一阶模型 τ ≈ 30 s，做 `T_real = T_meas + τ · dT/dt`。

## 6. 误差分析（v2 细化，全面 JCGM 化）

### 6.1 不确定度来源汇总表

| 量 | 类型 | 分布 | 标准不确定度 u | 来源 |
|----|------|------|----------------|------|
| T₁, T₂ | B | 矩形→σ=a/√3 | 0.087 ℃ | Pt100 A 级 ±0.15 ℃ |
| P₁, P₂ | B | 矩形 | 0.58 kPa | ±0.1% FS × 1 MPa |
| ΔP | B | 矩形 | 0.087 kPa | ±0.075% FS × 200 kPa |
| Q₀（主管电磁） | B | 矩形 | 0.29% | ±0.5% |
| Qᵢ（支路超声） | B | 矩形 | 1.15% | ±2% |
| 安装偏差（超声直管段） | B | 三角 | 1.0% | 现场保守估计 |
| 动态延迟（温度） | A | 正态 | 由反卷积残差估 | 约 0.2 ℃ |

### 6.2 MC 误差传递（亮点 3）

- 输入：上表中每一量，按对应分布生成 N=10000 采样。
- 正向模型：HII = f(Q₀, Qᵢ, ΔP, ΔT_i)
- 输出：HII 的 PDF → 95% 置信区间 [HII_lo, HII_hi]
- 判定规则修正：
  - 若 HII_hi < 0.10 → 可靠正常
  - 若 HII_lo > 0.20 → 可靠中度失调
  - 若区间跨越阈值 → **标记为"判定置信不足"**，延迟 5 min 再判。

实现：`MetroloPy` 或 40 行 numpy 循环即可。

### 6.3 典型传递结果（预估）

| 真实 HII | 仪表误差贡献下 HII 95% CI | 判定 |
|----------|---------------------------|------|
| 0.05 | [0.02, 0.08] | 正常 ✓ |
| 0.15 | [0.11, 0.19] | 轻度 ✓ |
| 0.11 | [0.08, 0.14] | **跨越阈值，延迟判定** |
| 0.30 | [0.25, 0.35] | 中度 ✓ |

这比单一阈值判定**更工程化**，也是答辩时的高分点。
